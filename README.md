# ⛪ CDMC Las Piñas Portal

A dedicated digital hub for the **Christian Disciples of Missionary Church (CDMC)** community in Las Piñas, Philippines. This portal is designed to provide easy access to church news, daily scriptures, and a direct line for prayer requests.

## ✨ Features
*   **🙏 Prayer Wall**: A seamless integration using Google Forms that allows members to submit prayer concerns directly to church leadership.
*   **📖 Daily Verse**: Features a "Verse of the Day" and weekly reading plans to keep the congregation engaged with scripture.
*   **📢 Church Updates**: Real-time information on upcoming events like Family Day and VBS Kalinga.
*   **🌅 Dynamic Greetings**: Personalized time-based greetings (Good Morning/Afternoon/Evening) synced to Manila, Philippines time.

## 🛠️ Tech Stack
*   **Language**: Python 3.x
*   **Framework**: [Streamlit](https://streamlit.io/)
*   **Database/Storage**: Google Sheets via Google Forms
*   **Deployment**: Streamlit Cloud
